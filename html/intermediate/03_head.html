title 태그
브라우저의 구성중 탭의 제목을 설정(곧 그 문서의 제목)
meta 태그
>
브라우저가 출력하는 문서의 기능, 컨텐츠, 환경, 스타일링, 자바스크립트 등을 설정
  • charset 속성 : 문서의 문자 인코딩
  • http-equiv 속성 : 문서를 전송받아 해석하는 브라우저의 멀티미디어 형식을 설정
  • meta name="viewport" : 문서의 출력 형태를 설정
  • content="width=device-width, initial-scale=1.0" : 현재 사용하는 기기의 출력 배율 1을 사용
  • 위의 viewport, content설정은 반응형 웹을 제작시 필수!!!
  • name="keywords" : 문서가 포함하고 있는 컨텐츠들의 대표 키워드
  • name="author" : 문서 작성자
  • name="description" : 문서가 포함하고 있는 컨텐츠의 주제
style tag
CSS 코드를 작성해서 문서가 로드 될 때 실행될 수 있도록 하는 부분
script tag
자바스크립트 코드를 작성해서 문서가 로드 될 때 실행될 수 있도록 하는 부분
link tag
이 문서가 필요한 리소스가 있는 위치값(url)을 문서가 로드될 때 실행하여 문서에 적용될 수 있도록 함
base tag
문서가 링크(a tag)를 사용할 때 절대적인 시작 주소를 설정
